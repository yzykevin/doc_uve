# 进程管理

## -show_z

显示属于当前用户的所有僵尸进程。

- **默认**：`False`
- **用法**：使用此选项列出当前用户的所有僵尸进程。

## -kill_z

杀死属于当前用户的所有僵尸进程。

- **默认**：`False`
- **用法**：使用此选项终止当前用户的所有僵尸进程。

## -show_grep

使用 `grep` 显示与特定模式匹配的进程。

- **类型**：`string`
- **默认**：空字符串
- **用法**：指定一个模式来过滤进程。示例：`-show_grep=xxx`。

## -kill_grep

使用 `grep` 杀死与特定模式匹配的进程。

- **类型**：`string`
- **默认**：空字符串
- **用法**：指定一个模式来终止进程。示例：`-kill_grep=xxx`。
